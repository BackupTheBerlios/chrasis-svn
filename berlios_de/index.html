Please help us generate new character stroke data! Steps:

  1. Get CharacterCreater: svn co svn://svn.berlios.de/chrasis/Linux/CharacterCreater/trunk
  2. Compile it: cd trunk && ./autogen.sh && ./configure && make && make install
  3. RUN!: charactercreater
  4. write characters, save the file, and send it to me... "palatis _AT_ gmail _DOT_ com".
